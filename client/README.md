### Installation

- Clone the repository
  ```sh
  git clone https://github.com/travolgi/qrcode-generator.git
  ```
- Navigate to repository folder
  ```sh
  cd qrcode-generator
  ```
- Install npm packages
  ```sh
  npm install
  ```
- To run the app in the development mode run in the project directory: 
  ```sh
  npm start
  ```
